# Online Store (C++ Backend)

**Description**  
This project implements the backend of an online store using **C++**, using custom libraries and file-based storage. It manages core functionalities such as product management, customer authentication, order processing, and inventory tracking.

---

## Features

- Add, update, delete, and list products  
- Customer registration and login  
- Order creation and processing  
- Inventory management using file storage  
- Modular code design with separate libraries for different functionalities  
- Input validation and error handling for reliable operations
